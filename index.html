UWAFT ECO Car Hackathon — 1st Place

University of Waterloo · MATLAB/Simulink · YOLO · Computer Vision · AI/ML

Built the full perception-to-control loop that let our autonomous taxi behave like a real "rules of the road" vehicle inside MATLAB/Simulink.

🔍 Computer Vision — Stop-Sign Detection (ACF)

Trained an ACF (Aggregated Channel Features) stop-sign detector in MATLAB using ground-truth bounding boxes. The trained model outputs a bounding box (where it believes the stop sign is) along with an ACF score — a match-strength value indicating how strongly that region matches learned stop-sign features. I validated the detector using held-out images and precision/recall-style evaluation, then tuned the detection threshold to balance missed detections vs. false positives. The score gap between real stop signs and false positives was used to set a reliable cutoff.

🤖 AI/ML — Traffic-Light Detection (YOLO)

To extend perception beyond stop signs, I trained a YOLO-based traffic-light detector in Python using Google Colab GPUs (CUDA) to classify traffic-light states (red / yellow / green). On validation the model achieved:

  • 88.7% Precision — high correctness of detections
  • 86.2% Recall — strong coverage of actual objects
  • 91.3% mAP@50 — robust overall detection performance

This gave us a modern deep-learning perception component that complements the lightweight ACF approach, evaluated using standard object-detection metrics rather than visual inspection alone.

⚙️ Controls Integration — Simulink SpeedController

Integrated perception outputs into the Simulink control stack by converting detections into a desired-speed override that feeds the existing SpeedController. Rather than driving motors directly from vision, the perception layer produces a clean "stop / go" decision (based on detection confidence, thresholds, and stability), and the controller handles smooth deceleration to 0 m/s and resumption to cruise speed.

This separation kept the system modular: perception answers what is present, while control answers how the vehicle should respond — resulting in predictable behavior and higher trust in the vehicle's actions during live demos.

Hackathon Gallery

Baggage Bot

An automated package delivery system designed to streamline airport loading/unloading operations. The bot operates in two modes—infinite delivery or a set number of packages—and uses ultrasonic sensors for collision avoidance and a color sensor to determine the path to follow based on the sticker color on each package.

The bot's path is monitored before each movement. If an obstruction persists for over 30 seconds, the bot executes the goHome function to return to its original location. Otherwise it awaits user input to resume.

This project was the reason I became interested in software. From the late nights fixing the code to the final demo, I fell in love with the entire development process. The bot runs on an Arduino Mega with custom C++ libraries for sensor fusion, PID speed control, and a task scheduler that prioritizes safety checks over movement commands.

Housing Prices Prediction

An end-to-end ML pipeline for predicting housing prices using pandas, numpy, matplotlib, seaborn, and scikit-learn. Covers the full workflow: data loading and cleaning, exploratory data analysis with visualizations, feature engineering, and model training with Linear Regression and Random Forest Regressor. Model performance is evaluated using R-squared, with Random Forest further optimized through Grid Search.

Through this project I strengthened my skills in Python, data manipulation, visualization, and machine learning—developing a deeper understanding of end-to-end pipelines and their importance in building accurate, optimized models.

Digit Recognition

A deep learning pipeline classifying handwritten digits using the MNIST dataset. A CNN with two convolutional layers, dropout, and two fully connected layers is trained with Adam optimizer and CrossEntropyLoss—delivering 99%+ accuracy while demonstrating how to productionize model evaluation with experiment tracking and augmentation hooks.

This project honed my Python programming, deep learning techniques, and data visualization skills, providing hands-on experience in managing datasets, architecting neural networks, and fine-tuning deep learning models end to end.
